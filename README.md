## 📌 About Project

Bharat-Insight is a web application that:
- Shows large datasets in a table
- Allows filtering and searching data
- Supports multiple departments (like Health, Agriculture)
- Uses AI to give insights from data

---

### 🌐 Landing Page

- Modern UI layout

### 📊 Dashboard

## ⚙️ How to Run

1. Clone the repo
```bash
git clone https://github.com/01-meghal/bharat-insight.git
```

2. Install dependencies
```bash
npm install
```

3. Setup Environment Variables
Create a `.env.local` file in the root directory and add your Gemini API Key:
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

4. Run the development server
```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).
